# CatBot

A Desktop Agent App. Like OpenClaw, but with zero setup required.

![CatBot LOGO](https://github.com/wellCh4n/catbot/blob/main/resources/icon.png "CatBot Logo")




## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
